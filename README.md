[ 管理ツール ]  
---
DBに登録されているデータの閲覧や管理ツールにアクセスするアカウントの管理が出来ます。  

[ メインとなるAPI ]  
---
ユーザー情報の取得・更新  
救難信号の募集登録・一覧取得・更新  
ゲストの登録・募集に参加しているゲストの取得・更新
